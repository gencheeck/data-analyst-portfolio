# Данные

Датасеты в репозиторий не входят. Если хотите прогнать ноутбуки локально,
положите файлы сюда с такими именами:

- `data.csv`
- `users_go.csv`, `rides_go.csv`, `subscriptions_go.csv`
- `real_estate_data.csv`
- `mkrf_movies.csv`, `mkrf_shows.csv`
- `telecomm_csi.db`
- `telecom_clients.csv`, `telecom_dataset.csv`
- `visits_info_short.csv`, `orders_info_short.csv`, `costs_info_short.csv`
- `moscow_places.csv`, `admin_level_geomap.geojson`
- `vacancies_da.xlsx`, `vacancies_sa.xlsx`

Часть учебных файлов лежит на `https://code.s3.yandex.net/datasets/<имя>`.
Перед скачиванием лучше глянуть условия использования источника.
